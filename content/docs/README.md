# Tài liệu project

Thư mục này tập trung các tài liệu Markdown/HTML để dễ quản lý và tra cứu.

## Cấu trúc

- `analysis/strategic`: báo cáo chiến lược tổng quan.
- `analysis/reports`: báo cáo phân tích theo chủ đề.
- `analysis/html`: các báo cáo định dạng HTML.
- `resources`: mẫu biểu và tài liệu tham khảo.
- `skills`: tài liệu kỹ năng/đặc tả tác vụ.
